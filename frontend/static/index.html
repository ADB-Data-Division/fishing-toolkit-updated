Welcome to ADB's Fishing Toolkit!

ADB's Fishing Toolkit helps analyze how tropical cyclones affect fishing activity. By combining satellite boat detections and cyclone tracks, it provides clear insights on disruptions across fishing grounds.

Choose a mode of analysis to begin:

Nowcast Mode

Generates assessments using current or synthetic cyclone data to evaluate immediate effects on fishing grounds.

Specify the number of days to look back for cyclone data (default: 7 days)

Historical Mode

Provides comprehensive analysis of your cyclone events to understand long-term impacts on fishing activity.
